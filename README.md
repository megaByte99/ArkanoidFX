# ArkanoidFX
Arkanoid game written in JavaFX

KeyBoards:
<ul>
  <li>LEFT/RIGHT: Move the player left/right</li>
  <li>P: Pause the game</li>
  <li>F: Shoot lasers (when you take the powerup)</li>
  <li>F2: New game</li>
  <li>Space: Launch the ball</li>
</ul>
